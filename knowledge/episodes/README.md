# episodes —— 情節記憶（可回憶單元）

放「值得回憶的完整經歷」——一次做完某件事的完整現場：為何做、怎麼做、用了哪些概念、結果如何。它是 experience（蒸餾的教訓）背後的完整故事。

## 什麼進這裡
- 做完一個 roadmap 項，若它**重大、未來可能要回看「當初到底怎麼回事」** → 存一個 episode。
- 大部分做完的事**不存**——蒸餾成 experience 教訓就好，完整現場讓它淡去（情節大量衰退是正常的）。

## 與 experience 的關係
experience = 教訓（精華索引）；episodes = 現場（完整經歷）。一條 experience 教訓**連結**它的 episode——從教訓可回到現場。

## 檔名
`YYYY-MM-DD-做了什麼.md`：日期 + 一句它是什麼。
